WebSocketを使ったチャットアプリ
======================
WebSocketを使った簡易チャットアプリです。

ユーザ認証では次のプロバイダを利用できます。
- Facebook
- Github
- Google

## Usage
    $ go build -o go-chat
    $ ./go-chat -addr "localhost:8080"
